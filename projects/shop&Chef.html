Shop & Chef:

A Grocery List and Meal Planning App

About The Project

Shop and Chef is an application which enables users to easily create grocery shopping lists, either by adding items individually as needed, or by selecting them from recipe ingredients in the Shop & Chef database.

Users have the ability save their favorite recipes to their unique accounts. Recipes can be found by searching based on specific ingredients, types of meals, or dietary categories.

My Role

My role in the project was entirely in front end programming and designing UX/UI elements.

My design partner Luke Martin and I collaborated on planning the color scheme for the application.

Luke designed the logo, and I selected and placed imagery, laid out the structure of the pages, chose fonts, and connected front-end routes for API calls to the recipe database.

Technology Used

We used EJS Templating Engine to render our data as HTML Markup to be viewed on the front end. Through using EJS to create templates for our pages we were able to eliminate redundant code and create more streamlined, dry code for the application.

We used AJAX for all of our front end API calls.

Our styling was created using Bootstrap for our navbar, buttons, and user input fields. We used Bootswatch for our theme, and quite a bit of custom css written for placing the background image, font styling, custom color schemes and other fine tuning of the application's appearance. Included in our custom css styling is a google font to style all of the text in our application.
